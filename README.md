# Regular_PDE_Python
Two Python Functions that solve Regular PDEs Equations in Two Variables.

## Ex :
a\*Uxx + b\*Uyy + c\*Uxy + d\*Ux + e\*Uy + f\*U + <any equ in (x,y)> = 0
